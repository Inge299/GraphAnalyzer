# Nodex Brand Pack (Light Theme)

## Название
- Nodex

## Смысл
- Node + index/explore
- AI-помощник для анализа графов

## Основные цвета
- Primary Violet: `#5B3FD6`
- Soft Violet: `#8C7BEE`
- Accent Cyan: `#16C7D8`
- Deep Ink: `#221B47`
- Surface Light: `#F7F8FF`

## Ассеты
- `nodex-mark-light.svg` - основной знак (градиентный стиль)
- `nodex-logo-horizontal-light.svg` - базовый логотип для шапок и документов
- `nodex-wordmark-light.svg` - wordmark-вариант
- `nodex-app-icon-light.svg` - иконка приложения
- `nodex-cover-light.svg` - обложка для превью/соцсетей (1200x630)
- `icons/*.svg` - UI-иконки в стиле Nodex

## Рекомендации
- Для бренда по умолчанию использовать градиентный знак.
- Cyan использовать как акцент AI-узла/искра.
- Держать фон светлым (`#F4F2FF`...`#FFFFFF`) для стабильного контраста.
